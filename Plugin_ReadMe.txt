★対象Version
　びわっぽい Version 0.1.8.2以降

　これにより、使用しているOS(WindowsXP/Vista)によってはプラグインを更新しても表示できない可能性があります。
　表示できない場合はお使いのＯＳなどの情報を以下の掲示板に書き込みいただくと解決するかもしれません。

★プラグインのコンパイル
・コメントを非表示にしたい場合。
　同梱の「コメントなしプラグインコンパイル.bat」を実行してください。
・DMMのサイトIDをDMMa/DMMoに変換した場合。
　びわっぽい用補助ツール内の「DMM修正用プラグインコンパイル.bat」を実行してください。

上記以外の方はこれまで通り「事前準備：プラグインコンパイル.bat」を実行してください。

★関連ツール
　びわっぽい用補助ツール(LVtool.exe)
(1)びわっぽい起動時にエラーとなる設定ファイル(LiveViewer.setting)中の文字列を除去します。
(2)DMMお気に入りインポート／エクスポート時にエラーになるため設定ファイルを修正します。
(3)しんびわとのお気に入りインポート／エクスポートのためファイルを変換します。
※別途配布中。
