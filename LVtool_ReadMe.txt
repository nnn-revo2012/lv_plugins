琵琶っぽい設定ファイル修正ツール(LVtool.exe)

★概要
・琵琶っぽい起動時にエラーとなる設定ファイル(LiveViewer.setting)中の文字列を除去します。
・お気に入りインポート／エクスポート時にDMMが失敗するため設定を修正します。

★動作環境
・Microsoft Windows 7以降のＯＳ
・Microsoft .NET Framework 4.5以上

★ダウンロード
　実行ファイルは琵琶っぽい用プラグイン詰め合わせ(2019/03/28リリース)以降のzipファイルに同梱しています。
　修正ツールのリポジトリ（ソースファイル、変更履歴など）
  https://github.com/nnn-revo2012/LVtool
　※Visual Studio 2015 expressで開発しています。

★使い方

■設定ファイル(LiveViewer.setting)中のエラーとなる文字列除去

１．琵琶っぽいが起動中なら必ず終了してください。
２．LVtool.exeをダブルクリックします。
３．開いたら、文字列を除去したい設定ファイルをフォームにドラッグ＆ドロップします。
　　エラーになった設定ファイルは LiveViewer.setting.Error(日付) のような名前になっています。
４．「変換しました。」と表示されたら成功です。
５．変換した元のファイルは (元のファイル名).bak という名前になります。不要なら削除してかまいません。
６．変換したファイル名が LiveViewer.setting でない場合は LiveViewer.setting に名前を変更してください。
７．琵琶っぽいを起動してください。正常に起動すればＯＫです。



■お気に入りインポート／エクスポートのための設定修正。
※この修正は琵琶っぽいでお気に入りのインポート／エクスポートを行わない人には不要です。
※手順は今後変更になる可能性があります。

１．琵琶っぽいが起動中なら必ず終了してください。
２．LVtool.exeをダブルクリックします。
３．フォーム下にある「お気に入りインポート修正」のところのチェックをクリックしてチェックします。
４．設定ファイル(LiveViewer.setting)を変換します。設定ファイルをフォームにドラッグ＆ドロップします。
５．「変換しました。」と表示されたら成功です。
６．次にDMMのプラグインを修正します。Plugin_DMMa.csとPlugin_DMMo.csをフォームにドラッグ＆ドロップします。
７．「変換しました。」と表示されたら成功です。
８．ここで 事前準備：プラグインコンパイル.bat を実行してください。
９．琵琶っぽいを起動してください。正常に起動して、以下のように表示されていればＯＫです。
　　・設定→プラグインのサイト名で DMMa DMMo の様に表示されていること。
　　・一覧表示で■DMMa ■DMMo の様に表示されていること。
　　・お気に入り、ブラックリスト、履歴でサイトが DMMa DMMo の様に表示されていること。
　　・エクスポートした時 (DMMa)、(DMMo) の様に表示されていること。
１０．この修正後にエクスポートしたファイルはそのままインポート可能です。


★上記修正前にエクスポートした(DMM(a))となっているファイルを上記修正後にインポートしたい場合
　修正前にエクスポートしたファイルを以下の手順で変更してください。

１．琵琶っぽいが起動中なら必ず終了してください。
２．LVtool.exeをダブルクリックします。
３．フォーム下にある「お気に入りインポート修正」のところのチェックをクリックしてチェックします。
４．エクスポートしたファイルを変換します。エクスポートしたファイルをフォームにドラッグ＆ドロップします。
５．「変換しました。」と表示されたら成功です。
６．変換後のファイルはそのままインポート可能です。


★DMMのプラグインがアップデートされた場合
　お手数ですがDMMプラグインのみ以下の手順で変更してください（設定ファイルは変更の必要ありません）。
　※この手順は変更になる可能性があります。

１．琵琶っぽいが起動中なら必ず終了してください。
２．LVtool.exeをダブルクリックします。
３．フォーム下にある「お気に入りインポート修正」のところのチェックをクリックしてチェックします。
４．DMMのプラグインを修正します。Plugin_DMMa.csとPlugin_DMMo.csをフォームにドラッグ＆ドロップします。
５．「変換しました。」と表示されたら成功です。
６．ここで 事前準備：プラグインコンパイル.bat を実行してください。
７．琵琶っぽいを起動してください。正常に起動して、以下のように表示されていればＯＫです。
　　・設定→プラグインのサイト名で DMMa DMMo の様に表示されていること。
　　・一覧表示で■DMMa ■DMMo の様に表示されていること。
