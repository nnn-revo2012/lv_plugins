びわっぽい用補助ツール(LVtool.exe) Ver 0.23

★概要
・びわっぽい起動時にエラーとなる設定ファイル(LiveViewer.setting)中の文字列を除去します。
・DMMお気に入りインポート／エクスポート時にエラーになるため設定ファイルを修正します。
・しんびわとのお気に入りインポート／エクスポートのためファイルを変換します。

★動作環境
・Microsoft Windows 7以降のＯＳ
・Microsoft .NET Framework 4.5以上

★使い方

■設定ファイル(LiveViewer.setting)中の起動時エラーとなる文字列除去

１．びわっぽいが起動中なら必ず終了してください。
２．LVtool.exeをダブルクリックします。
３．開いたら、文字列を除去したい設定ファイルをフォームにドラッグ＆ドロップします。
　　エラーになった設定ファイルは LiveViewer.setting.Error(日付) のような名前になっています。
４．「変換しました。」と表示されたら成功です。
５．変換した元のファイルは (元のファイル名).bak という名前になります。不要なら削除してかまいません。
６．変換したファイル名が LiveViewer.setting でない場合は LiveViewer.setting に名前を変更してください。
７．びわっぽいを起動してください。正常に起動すればＯＫです。


■DMMお気に入りインポート／エクスポートのための設定修正
※この修正はびわっぽいでお気に入りのインポート／エクスポートを行わない人は不要です。

１．びわっぽいが起動中なら必ず終了してください。
２．LVtool.exeをダブルクリックします。
３．メニュー→ツール→DMMお気に入りインポート用修正の順に選びます。
４．フォーム下にある「DMMお気に入りインポート用修正」のところのチェックをクリックしてチェックします。
５．設定ファイル(LiveViewer.setting)を変換します。設定ファイルをフォームにドラッグ＆ドロップします。
６．「変換しました。」と表示されたら成功です。
７．次にDMMのプラグインをコンパイルし直します。
　Plugin_DMMa.csとPlugin_DMMo.csは2019/4/2以降のバージョンを使用してください。
８．DMM修正用プラグインコンパイル.bat を実行してください。
９．エラーが出ないでコンパイルできれば成功です。
１０．琵琶っぽいを起動してください。正常に起動して、以下のように表示されていればＯＫです。
　　・設定→プラグインのサイト名で DMMa DMMo の様に表示されていること。
　　・一覧表示で■DMMa ■DMMo の様に表示されていること。
　　・お気に入り、ブラックリスト、履歴でサイトが DMMa DMMo の様に表示されていること。
　　・エクスポートした時 (DMMa)、(DMMo) の様に表示されていること。
１１．この修正後にエクスポートしたファイルはそのままインポート可能です。

★DMMのプラグインがアップデートされた場合
　お手数ですが以下の手順で変更してください（設定ファイルは変更の必要ありません）。

１．琵琶っぽいが起動中なら必ず終了してください。
２．LVtool.exeと同梱のDMM修正用プラグインコンパイル.bat を実行してください。
３．エラーが出ないでコンパイルできれば成功です。
４．琵琶っぽいを起動してください。正常に起動して、以下のように表示されていればＯＫです。
　　・設定→プラグインのサイト名で DMMa DMMo の様に表示されていること。
　　・一覧表示で■DMMa ■DMMo の様に表示されていること。


■びわっぽい、しんびわのインポート／エクスポートのためのファイル変換

１．LVtool.exeをダブルクリックします。
２．開いたら、インポートまたはエクスポートしたいファイルをフォームにドラッグ＆ドロップします。
３．「変換するデータの種類を選んでください。」と表示されます。
　「びわっぽい→しんびわ」はびわっぽいでエクスポートしたファイル(Export.txt)をしんびわ用に変換します。
　「しんびわ→びわっぽい」はしんびわのdefine.txtをびわっぽい用に変換します。
　「びわっぽい→びわっぽい」はびわっぽいでエクスポートしたファイル（DMM(a)の形式のみ）をDMMaの形式に変換します。
４．選んだら右の「ＯＫ」をクリックしてください。中止する場合は「キャンセル」をクリックしてください。
５．「お気に入りファイルを変換しました。」と表示されたら成功です。
６．変換したファイルは (元のファイル名)_日付_時間.txt という名前になります。
７．特にしんびわ用のdefine.txtを間違って消さないように注意してください。
　　また、簡単な動作テストは行っておりますが、本ソフトでしんびわの動作を完全に保障するものではありません。
